<H1>Задание 1 (1.1)</H1>
<A NAME="chap1.2"></A>
Обобщенная коробка.<br>
Создайте сущность Коробка, которая обладает следующими характеристиками:<br>
 Может хранить один произвольный объект в один момент времени.<br>
 Объект можно получить и разместить на хранение в любой момент времени.<br>
 Если объект забирают из коробки – ссылку на этот объект необходимо обнулить.<br>
 Если объект кладут в коробку, но она не пуста – необходимо выкинуть исключение.<br>
 Имеет метод проверки на заполненность.<br>
 Методы класса должны работать с тем типом данных, который был указан во время
создания объекта<br>
Создайте Коробку которая может хранить целочисленное значение, разместите туда число 3.
Передайте Коробку в какой-либо метод, извлеките значение, и выведите его на экран.<br>
<br>
Реализация:<br>
Создаётся обобщённый класс Box  заданный типом T, который может хранить один объект произвольного типа.
В классе объявляется приватное поле T filling. При создании коробки оно null.
Метод putObject кладёт объект в коробку, но только если в коробке ещё ничего не лежит.
Если она занята, то выбрасывается исключение.
Метод takeFilling возвращает объкт и обнуляет его ссылку, для того чтобы коробка стала опять пустой.
Метод full проверяет содержит ли коробка объект.
В примере создаётся коробка с типом Integer, и кладётся туда число 3, затем показывается что
коробка заполнена. Для наглядности работы программы пытаемся положить туда ещё одно число = 5.
Программа показываает что коробка занята и выводится исключенние.
Далее коробка передаётся в метод processBox в main, там она проверяется на полноту, далее
вызывается takeFilling, извлекаетсяи объект и выводит его на экран, ссылка на этот объект обнуляется.
